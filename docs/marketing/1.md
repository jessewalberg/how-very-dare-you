## 11. Metrics & Monthly Review

### 11.1 Dashboard Setup
- [ ] `P1` Set up a simple metrics dashboard (Google Sheet or Notion) tracking:
  - Organic search traffic (Google Search Console + GA4)
  - Number of indexed pages (Search Console)
  - Email subscribers (email platform)
  - Social media followers (per platform)
  - Domain authority (Moz/Ahrefs free tools — check monthly)
  - Premium conversion rate
  - Bounce rate on advisory pages
  - Average pages per session

### 11.2 Monthly Review (1 hour, recurring)
- [ ] `P1` Schedule a recurring monthly calendar event for metrics review
- [ ] Each month, answer:
  - [ ] Which advisory pages get the most organic traffic? Create more content around similar titles.
  - [ ] Which keywords are ranking on page 2? Optimize those pages (biggest quick wins).
  - [ ] Which social posts drove the most site visits? Double down on that format.
  - [ ] What's the email open rate? Below 30% → test subject lines. Above 40% → great.
  - [ ] Are paid ads profitable? Calculate cost per signup and cost per premium conversion.

### 11.3 6-Month Targets
- [ ] Organic search traffic: 2,000–5,000 visits/month
- [ ] Indexed pages: 500+
- [ ] Email subscribers: 500–1,000
- [ ] Social media followers: 1,000+ across platforms
- [ ] Domain authority: DA 15–20
- [ ] Premium conversions: 2–5% of registered users
- [ ] Bounce rate on advisory pages: under 60%
- [ ] Average advisory pages per session: 2.5+
